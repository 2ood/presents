# Presents

선물을 주고받는 새로운 방법을 제시한다. 

* 선물을 Wish 할 수 있고
* 선물을 Merge 할 수 있고
* 받은 선물들을 자랑/감사표시 할 수 있고
* 이 모두를 aesthetic 하게 볼 수 있다.
